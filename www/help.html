Anders, als Du es von Knuddels.de bisher gewohnt bist, haben wir uns bei ChannelControl 3 für den PUNKT als Befehlseinleitung entschieden. Zum Beispiel .profile
Das hört sich zu Anfang suspekt an, aber nach kurzer Eingewöhnungszeit wird Dir das nichts mehr ausmachen. Im Gegenteil - sofern Du Knuddels regelmäßig am Smartphone nutzt, wirst Du die Vorzüge des PUNKT-Operators schnell erkennen.
Ein weiterer Vorteil ist: Durch diese kleine Änderung können wir Dir jederzeit kurze und verständliche Befehlsnamen ermöglichen, ohne in Konflikt mit knuddelseigenen Befehlen zu geraten.

Während Deines Aufenthalts errechnen wir stetig Dein persönliches Aktivitätslevel. Dieses Aktivitätslevel setzt sich aus vielen verschiedenen Faktoren zusammen. Welche dies sind, können wir Dir leider nicht verraten.
Dieses Aktivitätslevel soll es dem Channel-Besitzer ermöglichen zu erkennen, wie aktiv ein User ist.
Ebenso kann mithilfe dieses Levels eine Hürde für bspw. eine CM-Wahl gesetzt werden.

Es gibt verschiedene NICK-Icons, deren Bedeutung wir hier erläutern möchten.
Dieses Icon zeichnet den Channel-Besitzer aus.
Dieses Icon zeigt, dass ein Nutzer diesen Channel zu seinem Lieblingschannel gemacht hat.

  • Allgemeine Befehle
  • .donate ANZAHL
    Du spendest ANZAHL Knuddel an den Bot.
  • .help
    Ruft diese Hilfe auf.
  • .newsletter
    Ändert Deine Newsletter-Einstellungen.
  • .privacy
    Zeigt die Datenschutzerklärung an.
  • .profile [NICK]
    Ruft das Profil von NICK auf. Falls kein Name angegeben wurde, wird das eigene Profile angezeigt.
  • .system
    Listet allgemeine Information zur aktuellen Installation auf.
  • CM-Befehle
  • .applock
    Zeigt eine Liste aller gesperrten User an.
  • .applock NICK:GRUND*TAGE
    Sperrt NICK für TAGE (0 = Permanent) mit GRUND für das weitere Betreten des Channels.
  • .applock !NICK
    Entsperrt NICK wieder für den Channel.
  • .appmute
    Zeigt eine Liste aller User mit einem App-Mute an.
  • .appmute NICK:GRUND*MINUTEN
    Setzt NICK für MINUTEN (0 = 24 Stunden) mit GRUND auf die Mute-Liste.
  • .appmute !NICK
    Entfernt NICK wieder von der Mute-Liste
  • .teamchat TEXT
    Sendet eine Nachricht an alle anwesenden Teammitglieder des Channels.
  • MOD-Befehle
  • .bot mute
    Stellt den Bot ruhig.
  • .bot say:TEXT
    .bot say:TEXT
  • .bot do:TEXT
    .bot do:TEXT
  • .topic delete
    Löscht das Channelthema vollständig.
  • .topic preview:TEXT
    Zeigt eine Vorschau von TEXT als Channelthema
  • .topic set:TEXT
    Ersetzt das aktuelle Channelthema mit TEXT
  • .topic expand:TEXT
    Fügt TEXT an das Ende des aktuellen Channelthemas.
  • AppManager/Eigentümer-Befehle
  • .acp [modul]
    Öffnet das ControlPanel der App. Wenn [modul] angegeben, wird direkt die Konfiguratioon von [modul] aufgerufen.
  • .blacklist
    Zeigt eine Liste aller Nutzer an, die auf der Blacklist stehen.
  • .blacklist !NICK
    Entfernt NICK von der Blacklist
  • .blacklist NICK
    Setzt NICK auf die Blacklist
  • .knuddel
    Der Bot verschenkt Knuddel an einen zufälligen User.
  • .news THEMA:TEXT*STATUS
    Sendet einen Newsletter an alle User mit STATUS. Möglich: All, Family, Stammi, Team, Ehrenmitglied, Admin
  • .ownermsg
    (De)Aktiviert den Hinweis über wichtige Befehle beim Betreten des Channels.
  • .payout ZAHL
    Der Bot zahlt Dir ZAHL Knuddel auf Deinen Knuddelaccount.
  • .payout ZAHL:NICK[:GRUND]
    Der Bot zahlt ZAHL Knuddel auf NICKs Knuddelaccount. Die Angabe von GRUND ist optional.
  • .upgrade
    ChannelControl 3-Upgrade kaufen.
